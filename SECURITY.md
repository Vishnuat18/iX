## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it immediately.

**How to report:**
Please send an email to **security@ix-platform.com** (or open a private issue if applicable). 

**What to expect:**
- You will receive an acknowledgment of your report within 48 hours.
- We will provide a timeline for a fix within 7 days.
- Once fixed, we will coordinate a public disclosure if necessary.

> [!IMPORTANT]
> If you accidentally committed secrets (like API keys), please rotate them immediately and follow the instructions in the project documentation for secret removal.
